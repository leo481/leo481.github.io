创建自定义链

日常应用中默认的五条链已经基本满足需要了,对于某些中大型的应用,或者入站规则,则需要一种集中管理的机制,此时自定义链可以解决此类问题,下面简要介绍用法

  1. 创建自定义链

  2. 引用自定义链

  3. 重命名自定义链

  4. 删除自定义链

删除自定义链需要满足两个条件

  • 自定义链没有被引用,使用iptables -D 序号
  • 自定义链中没有任何规则,使用 iptables -t filter -F WEB
